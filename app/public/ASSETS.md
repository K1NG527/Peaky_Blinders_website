# Asset Files Guide

Replace any asset below with your own version (same filename, same format).

## Splash / Intro Videos
- **`thomas_intro.mp4`** — Splash screen video for Thomas Shelby's POV
- **`luca_intro.mp4`** — Splash screen video for Luca Changretta's POV

## Character Silhouettes (Hero Background)
- **`thomas_silhouette.png`** — Thomas Shelby silhouette shown in hero section
- **`luca_silhouette.png`** — Luca Changretta silhouette shown in hero when switched

## Backgrounds
- **`bg_texture.jpg`** — Main background texture (dark, grungy)
- **`luca_bg_texture.jpg`** — Background texture for Luca (currently same as Thomas, swap to customize)

## Props / Decorations
- **`peaky_cap.png`** — Floating 3D element (Peaky cap)
- **`gold_chain.png`** — Floating 3D element (gold chain)
- **`sidebar_chain.png`** — Sidebar decoration

## Supplier Images
- **`supplier_distillery.jpg`** — Camden Distillery photo
- **`supplier_imports.jpg`** — Solomon Imports photo
- **`supplier_pub.jpg`** — The Garrison Pub photo
 
 ## Cursor
 - **`cigar-realistic.svg`** — The cigar cursor image. Replace this with your own `cigar.png` or `cigar.gif` (update `CustomCursor.tsx` if filename changes).
 
 ## Audio
 - **`theme.mp3`** — Background music. Place your file in `/public` and update `App.tsx` to use it.
 
 ## Cinematic UI Sound Effects Pack
 Sounds are located in `/public/assets/sounds/`.
 
 - **`tick.mp3`** — Primary Click: Short, sharp metallic tick (Lighter flick / Revolver click).
 - **`paper_slide.mp3`** — Hover Layer 1: Soft brushed metal sweep (Sliding file).
 - **`vinyl.mp3`** — Hover Layer 2: Faint vinyl crackle texture.
 - **`heavy_clack.mp3`** — Nav Click: Heavier mechanical clack with industrial reverb (Hidden lever).
 - **`thomas_swell.mp3`** — Thomas Mode: Deep cello/brass swell.
 - **`knife_scrape.mp3`** — Luca Mode Layer 1: Knife-sharpen metallic scrape.
 - **`luca_drone.mp3`** — Luca Mode Layer 2: Low ominous drone.
 - **`page_turn.mp3`** — Ledger Page: Realistic book page flip / parchment rustle.
 - **`map_marker.mp3`** — Territory Map: Heavy stamp press / marker thud.
 - **`lamp_on.mp3`** — Desk Lamp On: Optimized ultra-short sharp snap.
 - **`lamp_off.mp3`** — Desk Lamp Off: Optimized ultra-short muted click.
